Remote Desktop — Moomoo Setup

Since Moomoo requires its OpenD gateway to run on a Windows PC, here are your best free options to access it remotely from your phone:

⭐ Chrome Remote Desktop

Easiest setup. Install on Windows, control from your phone browser or app. Works on any phone OS.

FREE EASIEST SETUP

AnyViewer

Best for unattended access — connect anytime without someone at the PC. Free for personal use, supports Android & iOS.

FREE UNATTENDED

Microsoft Remote Desktop

Built into Windows 10/11 Pro. Install the RD client on your phone. Most stable long-term option if your Windows is Pro edition.

FREE (WIN PRO)

🚀 Recommended Setup

1. Install AnyViewer on the Windows laptop
2. Set it to launch at startup
3. Keep Moomoo + OpenD running minimized
4. Use AnyViewer from your phone to interact with Moomoo when needed
